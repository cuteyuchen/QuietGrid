# 13. 测试与验收清单

## 1. 单元测试

### Scheduler

- [ ] 周末、节假日、半日市；
- [ ] 夏令时切换；
- [ ] 距盘前边界；
- [ ] 中途启动。

### Regime

- [ ] 数据不足返回 UNKNOWN；
- [ ] 波动扩张触发硬阻断；
- [ ] 低波动缓慢趋势不被误判为横盘；
- [ ] 双阈值避免抖动；
- [ ] 原因与分数一致。

### Grid

- [ ] 等比格位计算；
- [ ] 成本地板；
- [ ] 网格数量 clamp；
- [ ] 非等距扩张；
- [ ] tick/step/minNotional；
- [ ] 风险分配总和不超预算。

### Inventory

- [ ] 部分成交；
- [ ] 多 lot 配对；
- [ ] 净多/净空；
- [ ] 禁止增加同向库存；
- [ ] orphan position 恢复。

### Risk

- [ ] 会话止损；
- [ ] 窗口止损；
- [ ] 连续亏损熔断；
- [ ] 冷却恢复；
- [ ] 强制离场覆盖所有状态；
- [ ] 人工恢复不能绕过风险。

## 2. 集成测试

- [ ] 下单 → 接受 → 部分成交 → 完全成交 → 补单；
- [ ] POST_ONLY 拒单；
- [ ] 下单超时但交易所已接受；
- [ ] WebSocket 漏事件后 REST 对账；
- [ ] 条件止损创建、更新和取消；
- [ ] 进程重启恢复；
- [ ] SQLite 写入失败；
- [ ] API 控制命令审计。

## 3. 回测验收

- [ ] LookAheadViolation 自动检测；
- [ ] 成本、资金费和止损滑点；
- [ ] 样本内与样本外分开；
- [ ] Walk-Forward；
- [ ] 参数平台稳定性；
- [ ] Maker 成交率敏感性；
- [ ] 极端窗口压力测试；
- [ ] Monte Carlo P5 结果仍在可承受范围。

## 4. 故障注入

- [ ] REST 断开；
- [ ] WS 断开 1/5/30 分钟；
- [ ] 时间不同步；
- [ ] 数据过期；
- [ ] 交易所返回重复成交；
- [ ] 数据库锁；
- [ ] 进程在持仓中被 kill；
- [ ] 止损单被外部取消；
- [ ] 费率从 0 变为非 0。

## 5. 前端验收

- [ ] TESTNET/LIVE 环境醒目；
- [ ] 所有风险状态不只依赖颜色；
- [ ] 危险操作二次确认；
- [ ] 恢复请求被风控拒绝时说明原因；
- [ ] SSE 断开回退轮询；
- [ ] API 401/403/500 正确处理；
- [ ] 移动端可紧急查看和关闭；
- [ ] 参数差异与版本可见。

## 6. 小资金实盘验收

- [ ] 单账户；
- [ ] 单标的；
- [ ] 1 倍有效杠杆；
- [ ] 可承受完全损失的极小资金；
- [ ] 至少运行多个独立窗口；
- [ ] 每窗口结束无挂单、无持仓；
- [ ] 实际成交率、滑点和资金费与回测对比；
- [ ] 无人工违反系统规则；
- [ ] 扩大资金前形成书面评审。

## 7. 发布门槛

任何以下情况都禁止扩大资金：

- 样本外为负；
- 最坏窗口超过风险预算；
- 实盘成交显著差于回测；
- 发生无法解释的仓位或订单差异；
- 风控曾被人工绕过；
- Web 未鉴权或止损兜底未验证；
- Maker 费率无法确认。
